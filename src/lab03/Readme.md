# О проекте
## Описание класса контейнера ProductCatalog:
Класс ProductCatalog — это контейнер для хранения и управления объектами класса *[Product](../lab01/model.py)* (созданного в первой лабораторной работе).

Он обеспечивает безопасное управление коллекциями: добавление, удаление, поиск по имени, сортировка и фильтрация объектов.
### Основные характеристики:
* Управление — Добавление, удаление, доступ по индексу
* Поиск — По названию
* Сортировка — По цене
* Фильтрация — Получение подколлекций (в наличии, сняты с производства, дорогие, дешевые)
* Хранение — Список объектов Product
* Итерация — Поддержка циклов
* Безопасность — Проверка типов, предотвращение дубликатов

## Методы коллекции
### Основные управленческие операции
* `add(item)` — добавить товар
* `get_all()` — получить список всех товаров

### Магические методы
* `__len__` — получение количества
* `iter` — итерация
* `__getitem__` — доступ по индексу
* `__str__`— Удобное представление информации о продукте, понятное пользователю

### Методы удаления и поиска
* `remove(item)` — удалить товар
* `remove_at(index)` — удалить по индексу
* `find_by_name(name)` — Поиск по имени
* `find_by ()` — Универсальный поиск (по названию, цене, модели и т. д.) и фильтрация (например, активные и неактивные)

### Методы сортировки и фильтры
* `sort_by_price()` — Расположите товары по цене в порядке убывания.
* `sort_by()` — Сортирует товары в соответствии с предпочтениями пользователя (по названию, цене, модели и т. д.)
* `get_active()` — Возвращает новую коллекцию, содержащую только активные товары.
* `get_deactive()` — Возвращает новую коллекцию, содержащую только Не активные товары.
* `get_expensive_products(min_price)` — дорогие товары
